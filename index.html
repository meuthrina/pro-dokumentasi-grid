Rows & Column

Mendefinisikan kolom / baris dengan cara menuliskan nilai dipisahkan oleh spasi. nilai merepresentasikan ukuran grid track dan spasi merepresentasikan grid line. 

            .container {
               display: grid;
               grid-template-columns: [track-size] ... | [line-name] [track-size] ... ;
               grid-template-rows: [track-size] ... | [line-name] [track-size] ... ;
            }

  • [track-size] = bisa berupa px | % | auto | fr (fraction).

  • [line-name] = nama bebas untuk track.

  • Explicit = ukuran (fix) yang telah kita tentukan nilainya.

  • Implicit = ukuran yang secara otomatis ditentukan oleh grid (auto).

                grid-template-rows: 50px 100px

Baris pertama kita atur ukuran fix nya yaitu 50px

Baris kedua kita atur ukuran fix nya yaitu 100px

Baris lain yang tidak kita atur ukuran fix nya (implicit), ukuran nya akan menyesuaikan isi dari konten/items nya.

Kolom pertama kita atur ukuran fix nya yaitu 90px

Kolom kedua kita atur ukuran fix nya yaitu 50px

Kolom ketiga kita atur ukuran fix nya yaitu 120px

Item 4, 5, 6 akan membuat baris baru karena hanya 3 kolom yang ditentukan nilai fix nya. Berbeda dengan baris, ukuran item pada properti column, akan menyesuaikan dengan ukuran content nya.

grid-auto-rows & grid-auto-columns

Mengatur ukuran grid track yang tidak dituliskan pada grid-template (implicit track). 

                grid-template-rows: 30px 60px 90px
                grid-auto-rows: 40px

Disini diketahui bahwa items yang diberi nilai fix adalah baris 1, 2, dan 3.

Karena baris 4, 5, dan 6 tidak kita tentukan nilai fix nya, maka nilai default nya ditentukan oleh grid (auto). Jika kita ingin mengubah ukuran auto tersebut maka kita menggunakan properti grid-auto-rows

1
2
3
4
5
6
                grid-template-rows: 50px 30px;
                grid-template-columns: 100px 50px;
                grid-auto-rows: 100px;

Disini diketahui bahwa items yang diberi nilai fix adalah item 1, 2, 3 dan 4.

Karena kita tahu jika kita menentukan jumlah sebuah kolom, maka kolom yang tidak kita tentukan nilainya akan membuat baris baru.

Pada kasus ini, item 5 dan 6 membuat baris baru karena kita hanya menentukan hanya 2 kolom yang diberi nilai fix.

Item 5 dan 6 tidak ditentukan nilai nya oleh grid-template-rows. maka jika kita ingin mengatur nilai sebuah item yang tidak kita tentukan nilai nya (implicit). maka kita memerlukan properti grid-auto-rows / grid-auto-column

grid-auto-flow

Mengatur penempatan item/cell pada grid track, termasuk yang ditulis secara implicit. (mirip dengan flex-direction) 

            .container {
               grid-auto-flow: row | column | row dense | column dense;
            }
          
                grid-template-columns: 100px 50px
                grid-auto-flow: column
                grid-auto-columns: 100px

Disini diketahui bahwa items yang diberi nilai fix adalah kolom 1 dan 2

Karena kolom 3, 4, 5, dan 6 tidak kita tentukan nilai fix nya, maka secara default kolom tersebut akan membuat baris baru.

Jika kita ingin mengatur supaya kolom yang tidak kita tentukan nilai fix nya tidak membuat baris baru, maka kita memerlukan properti grid-auto-flow

                grid-template-rows: 50px 30px
                grid-template-columns: 100px 50px
                grid-auto-flow: column

Disini diketahui bahwa items yang diberi nilai fix adalah item 1, 2, 3, dan 4

Karena item 5 dan 6 tidak kita tentukan nilai fix nya, maka seharusnya item tersebut akan membuat baris baru.

Karena kita menggunakan properti grid-auto-flow: column; , maka item 5 dan 6 mengubah arah direksinya yang seharusnya kebawah (membuat baris baru) menjadi ke arah kanan.

repeat()

Menentukan grid track secara berulang. 

                grid-template-columns: repeat(4, 1fr) repeat(5, 2fr)

Disini value pertama mendefinisikan untuk membuat 4 kolom dengan ukuran masing-masing 1 fraction.

Dan value selanjutnya menentukan kolom selanjutnya yaitu 5 kolom dengan ukuran masing-masing 2 fraction.

Walaupun jumlah items tidak sesuai dengan value kolom yang kita buat, yakni seharusnya 9 item. Tetapi, tetap ada ruang kosong yang dibuat untuk diisi item ke 9/selanjutnya.

auto-fill & auto-fit

Menentukan jumlah item untuk berada pada grid track.

  • auto-fit = menentukan akhir grid pada track terakhir.

  • auto-fill = menentukan akhir grid di tempat yang bisa diisi track lagi.

                grid-template-columns: repeat(auto-fit, 100px)

Disini value nya mendefinisikan untuk membuat kolom berukuran 100px yang jumlah kolomnya sebanyak besar ukuran (lebar) container nya.

Item yang tidak muat ketika viewport diperkecil akan berpindah ke bawah (membuat baris baru).

minmax()

Menentukan ukuran minimal & maksimal dari grid track. 

                grid-template-rows: minmax(100px, auto)
                grid-template-columns: minmax(auto, 50%) 1fr 3em

Fungsi minmax() menerima 2 argumen: yang pertama adalah ukuran minimum track dan yang kedua ukuran maksimum. Di samping value explicit, value nya juga bisa auto, yang memungkinkan track tumbuh / meregang berdasarkan ukuran konten.

Dalam contoh ini, track baris pertama diatur untuk memiliki tinggi minimum 100px, tetapi ukuran maksimum otomatis, dimana akan memungkinkan track baris untuk mengembangkannya konten lebih tinggi dari 100px.

Track kolom pertama memiliki ukuran minimum otomatis, tetapi ukuran maksimumnya 50%, dimana akan mencegahnya mendapatkan tidak lebih lebar dari 50% dari lebar kontainer grid.

row-gap and column-gap

mendefinisikan ukuran grid line / jarak antar baris atau kolom. 

            .container {
               row-gap: [line-size];
               column-gap: [line-size];
               gap: [row-gap] [column-gap]
            }
          
                row-gap: 20px
                column-gap: 5rem

Gap adalah properti untuk memberi jarak / ruang antar items baik secara horizontal maupun vertikal.

Value pada properti gap bisa menggunakan px, %, em, dsb.

1
2
3
4
                gap: 5px

Properti gap adalah shorthand untuk menuliskan 2 properti sekaligus, dengan value pertama yaitu [row-gap] dan value selanjutnya yaitu [column-gap].

Atau bisa juga ditulis 1 value pada properti gap, maka value tersebut akan berlaku untuk 2 value diatas.

place-items

Mensejajarkan items pada sumbu vertikal dan horizontal.

  • justify-items = mensejajarkan items pada sumbu horizontal.

  • align-items = mensejajarkan items pada sumbu vertikal.

  • place-items = shorthand untuk menuliskan 2 properti tersebut.

            .container {
               justify-items: start | end | center | stretch;
               align-items: start | end | center | stretch;
               place-items: [align-items] [justify-items]
            }
          
                justify-items: start

Item berada pada awal baris (sumbu horizontal).

                justify-items: center

Item berada pada tengah baris (sumbu horizontal).

1
                justify-items: end

Item berada pada akhir baris (sumbu horizontal).

1
                justify-items: stretch

Item akan memenuhi baris secara horizontal (ini adalah nilai defaultnya).

1
                align-items: start

Item berada pada awal kolom (sumbu vertikal).

1
                align-items: center

Item berada pada tengah kolom (sumbu vertikal).

1
                align-items: end

Item berada pada akhir kolom (sumbu vertikal).

1
                align-items: stretch

Item akan memenuhi baris secara vertikal.

1
                place-items: center end

properti ini adalah shorthand untuk menuliskan 2 properti sekaligus yakni [align-items] & [justify-items].

place-content

Mengatur posisi seluruh grid container pada sumbu vertikal & horizontal.

Ini bisa dilakukan ketika ukuran total grid lebih kecil dari ukuran containernya, biasanya ketika grid items nya menggunakan ukuran yang fixed (px).

  • align-content = mengatur seluruh grid container pada sumbu vertikal.

  • justify-content = mengatur seluruh grid container pada sumbu horizontal.

  • place-content = shorthand untuk menuliskan 2 properti tersebut.

            .container {
               align-content: start | end | center | stretch | space-around | space-between | space-evenly;
               justify-content: start | end | center | stretch | space-around | space-between | space-evenly;
               place-content: [align-content] [justify-content]
            }

self-item-properties

Mengatur posisi item secara individu menggunakan [align-self] & [justify-self]

  • align-self = mengatur posisi item pada sebuah cell terhadap sumbu vertikal (baris).

  • justify-self = mengatur posisi item pada sebuah cell terhadap sumbu horizontal (column).

  • place-self = shorthand untuk menuliskan 2 properti tersebut.

            .container {
               align-self: start | end | center | stretch;
               justify-self: start | end | center | stretch;
               place-self: [align-self] [justify-self]
            }
          
                .item-1 { align-self: start }
                .item-1 { align-self: center }
                .item-1 { align-self: end }

Grid Item Properties

Menentukan tempat dari grid item dengan mengacu pada nomor atau nama grid line yang spesifik.

  • grid-column-start & grid-column-end

  • grid-row-start & grid-row-end

  • [nomor] = mengacu pada nomor sesuai grid default, dengan pojok kiri atas adalah awal nilai positif. dan pojok kanan bawah adalah awal nilai negatif.

  • span [nomor] = ditambah berapa [nomor]

            .item {
               grid-row-start: [nomor] | span [nomor];
               grid-row-end: [nomor] | span [nomor];
               grid-column-start: [nomor] | span [nomor];
               grid-column-end: [nomor] | span [nomor];
            }
          
                grid-row-start: 2
                grid-row-end: 3
                grid-column-start: 2
                grid-column-end: 3

Diketahui bahwa garis awal dimulai dari garis pada baris ke 2, dan diakhiri di garis pada baris ke 3

Diketahui bahwa garis awal dimulai dari garis pada kolom ke 2, dan diakhiri di garis pada kolom ke 3

Sebenarnya, kita tidak perlu menuliskan grid-row/column-end pada keduanya, jika salah satunya sudah memiliki properti end.
